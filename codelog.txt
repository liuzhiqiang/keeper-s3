### 3. 主要功能说明
Keeper-S3 是一个专业的 WordPress 插件，用于：
- 将媒体附件上传到 Amazon S3 云存储
- 支持多种存储策略（本地、S3、双重存储）
- 提供批量迁移和转换功能
- 集成现代化 React 管理界面
- 支持多语言（中文、英文）

### 4. 技术架构
- 后端：PHP + WordPress 插件框架 + AWS SDK
- 前端：React + Material-UI + TypeScript
- 构建工具：WordPress Scripts + Webpack
- 依赖管理：Composer (PHP) + pnpm (Node.js)
